Using a watermarkÂ¶

A watermark is an image displayed behind the drawing. It is centered at all times (panning doesn’t affect it).

<div class="literally watermark"><canvas></canvas></div>

<script>
  $(document).ready(function() {
    var watermarkImage = new Image();
    watermarkImage.src = '/_static/sample_image.png';
    $('.literally.watermark').literallycanvas({
      backgroundColor: 'whiteSmoke',
      imageURLPrefix: '/_static/lib/img',
      watermarkImage: watermarkImage
    });
  });
</script>
